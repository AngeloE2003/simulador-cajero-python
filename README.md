# 🏧 Simulador de Cajero Automático Multifunción (Python)

Este proyecto implementa un sistema automatizado de cajero automático (ATM) desarrollado íntegramente en Python. Está diseñado para gestionar clientes, cajeros físicos y operaciones bancarias de manera eficiente, simulando un entorno financiero real con validaciones de seguridad estrictas.

## 🚀 Funcionalidades Principales
* **Gestión de Usuarios:** Registro, actualización y baja de clientes y cajeros.
* **Operaciones Bancarias:** Retiros, depósitos, transferencias y pagos de servicios.
* **Cálculo Financiero:** Algoritmo de desglose exacto de billetes para la dispensación de efectivo.
* **Seguridad y Auditoría:** Validación de credenciales y fondos, junto con un registro detallado de movimientos (logs).

## 🧠 Algoritmos y Estructuras de Datos Aplicadas
Para garantizar la escalabilidad y eficiencia del sistema, se implementaron soluciones algorítmicas avanzadas:
* **QuickSort:** Optimización del ordenamiento de registros con un desempeño promedio de O(n log n).
* **Búsqueda Binaria:** Localización rápida de datos y validación de registros, reduciendo la complejidad de las consultas a O(log n).
* **Estructuras de Datos:** Uso estratégico de listas y diccionarios para el acceso eficiente en memoria.

## 🛠️ Tecnologías
* **Lenguaje:** Python
* **Control de Versiones:** Git / GitHub
